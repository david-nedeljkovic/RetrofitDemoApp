# RetrofitDemoApp

## The application uses the Retrofit library to communicate with the API.
### API we are targeting: https://jsonplaceholder.typicode.com/
### Routes

### HTTP methods that are supported: 
#### GET/posts 
#### GET/posts/1
#### GET/posts/1/comments
#### GET/comments?postId=1
#### POST/posts
#### PUT/posts/1
#### PATCH/posts/1
#### DELETE/posts/1
